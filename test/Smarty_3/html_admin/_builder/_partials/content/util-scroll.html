Scroll & Scrollbar
Class Description Quick Example
.scrollable-vertical .scrollable-horizontal
.scrollable-styled-dark .scrollable-styled-light

.scrollable-[vertical|horizontal] allow scrolling long content. Can be mixed with .scrollable-styled-[dark|light] class.

.scrollable-styled-[light|dark] is a styled div scrollbar using CSS only (visible on mouse over, desktop only). We usualy use slimscroll plugin but the issue is overflow: hidden. Having a floating element outside of the container, will not be visible. And, of course, it's better using CSS than a javascript plugin. Of course, slimscroll is also available as a plugin but disabled by default.
